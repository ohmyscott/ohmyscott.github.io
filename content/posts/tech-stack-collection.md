## 前端技术

### [SVG.js](https://github.com/svgdotjs/svg.js)
- 🔧 **类型**：JavaScript 库
- 🌐 **链接**：[官网](https://svgjs.dev/) 
- 📖 **介绍**：一个轻量级的 JavaScript 库，用于操作和动画 SVG，无需任何依赖。提供了简洁的 API 来创建、操作和动画 SVG 图形。
- 🎯 **适用场景**：SVG 图形操作、数据可视化、动画效果、图表制作、矢量图形开发

### [HyperUI](https://github.com/markmead/hyperui)
- 🔧 **类型**：UI 组件库
- 🌐 **链接**：[官网](https://www.hyperui.dev/) | [GitHub Repository](https://github.com/markmead/hyperui)
- 📖 **介绍**：免费开源的 Tailwind CSS v4 组件集合，用于构建现代 Web 应用程序。从营销页面到管理后台，提供现成的 UI 组件来加速开发。无需安装、配置或设置，直接复制粘贴到你的 Tailwind CSS 项目中即可使用。
- 🎯 **适用场景**：快速原型开发、UI 组件库构建、营销页面设计、后台管理系统
