## 大模型

### [New API](https://github.com/QuantumNous/new-api)
- 🔧 **类型**：大模型聚合管理中转分发系统
- 📖 **介绍**：一个 AI 模型聚合管理中转分发系统，支持将多种大模型转为统一格式调用。支持 OpenAI、Claude、Gemini 等格式，可供个人或者企业内部管理与分发渠道使用。此项目基于 One API 进行二次开发。
- 🎯 **适用场景**：多模型统一管理、企业内部AI服务分发、API成本控制、模型调用代理

## 开发工具

### [Claude Code](https://github.com/anthropics/claude-code)
- 🔧 **类型**：AI 编程助手
- 📖 **介绍**：一个智能化的编程工具，存在于你的终端中，理解你的代码库，并通过自然语言命令帮助你更快地编码。它可以执行例行任务、解释复杂代码、处理 git 工作流程，并支持在终端、IDE 中使用，或在 GitHub 上 @claude。
- 🎯 **适用场景**：代码编写、技术博客撰写、代码审查、问题诊断、git 工作流程自动化、复杂代码解释

### [Claude Code Proxy](https://github.com/fuergaosi233/claude-code-proxy)
- 🔧 **类型**：API 代理服务器
- 📖 **介绍**：一个代理服务器，让 Claude Code 能够与 OpenAI 兼容的 API 提供商一起工作。它将 Claude API 请求转换为 OpenAI API 调用，允许你通过各种 LLM 提供商使用 Claude Code CLI。
- 🎯 **适用场景**：使用本地模型、替代API提供商、降低使用成本、自定义模型配置

## 效率工具

### [CrossDesk](https://github.com/kunkundi/crossdesk)
- 🔧 **类型**：远程桌面软件
- 🌐 **链接**：[官网](https://www.crossdesk.cn/)
- 📖 **介绍**：开源的跨平台远程桌面软件，支持硬件加速和 Web 访问。CrossDesk 是 MiniRTC 实时音视频传输库的实验性应用，具有网络透传、视频软硬编解码（H264/AV1）、音频编解码（Opus）、信令交互、网络拥塞控制、传输加密（SRTP）等基础能力。
- 🎯 **适用场景**：远程办公、技术支持、跨平台桌面访问、低延迟远程控制

## 在线工具

### [Tiny SVG](https://github.com/hehehai/tiny-svg)
- 🔧 **类型**：SVG 压缩工具
- 🌐 **链接**：[在线试用](https://tiny-svg.actnow.dev/) 
- 📖 **介绍**：一个开源的网页版 SVG 优化工具，专门用于优化 SVG 文件并生成特定框架的代码。支持在线使用，无需安装即可快速压缩 SVG 文件。
- 🎯 **适用场景**：网页性能优化、SVG文件压缩、前端开发、图标优化
